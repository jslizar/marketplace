# Structures to Avoid

## Em-dashes (top priority — kill all)

No em-dash anywhere. Not `—`, not `–`, not a spaced ` - ` standing in for one, not `--`.

| Found | Fix |
|-------|-----|
| "the substrate matters — not the sourcing" | Period or comma: "the substrate matters, not the sourcing" → better, drop the contrast: "the substrate matters." |
| "one engineer — not a 200-person team" | "one engineer does what a 200-person team used to." |
| "we shipped fast — and it worked" | "we shipped fast. It worked." |
| Aside set off by em-dashes "X — the part nobody admits — Y" | Use commas or parentheses, or split into two sentences. |

Method: if the em-dash joins two independent clauses, use a period. If it sets off an aside, use commas or parentheses. If it's doing a "not X, it's Y" reversal, see below and drop the negation.

## Binary Contrasts (top priority — the "not X, it's Y" family)

False drama through telegraphed reversal. Delete the negation, state the point.

| Pattern | Fix |
|---------|-----|
| "Not because X. Because Y." | "Y." |
| "[X] isn't the problem. [Y] is." | "Y is the problem." |
| "The answer isn't X. It's Y." | "The answer is Y." |
| "It's not about X, it's about Y." | "It's about Y." |
| "It feels like X. It's actually Y." | "It's Y." |
| "Not X. But Y." / "not X, it's Y" / "isn't X, it's Y" | "Y." |
| "stops being X and starts being Y" | "becomes Y" or just "is Y" |
| "doesn't mean X, but actually Y" | "means Y" |
| "not just X but also Y" | Name both plainly: "X and Y," or cut X. |

**Watt note:** Jared's brand *does* run real contrasts ("data company → signal company," "200-person data team → one Signal Engineer," "rent → own"). Those are positioning shifts stated as transformations, fine to keep. What dies is the *mechanical* "it's not X, it's Y" sentence shape used for rhetorical drama. Keep the idea, drop the formula: write "Watt is a signal company" not "Watt isn't a data company, it's a signal company."

## Negative Listing

Listing what something is *not* before revealing what it *is*. A rhetorical striptease.

- "Not a memo. Not a plan. A bet." → "It was a bet."
- "It wasn't X. It wasn't Y. It was Z." → "It was Z."

State the thing. The reader doesn't need the runway.

## False Agency

Giving inanimate things human verbs. AI loves this because it avoids naming the actor.

| Pattern | Problem |
|---------|---------|
| "the data tells us" | Data sits there. Someone read it and concluded. |
| "the market rewards" | Markets don't reward. Buyers pay for things. |
| "the complaint becomes a fix" | Someone fixed it. |
| "the decision emerges" | Someone decided. |
| "the culture shifts" | People changed behavior. |
| "the conversation moves toward" | Someone steered it. |

Name the human, or use "you" to put the reader in the seat. "Your agent reads 145,000 signals and picks the audience" beats "the data finds the audience."

## Passive Voice

Every sentence needs a subject doing something. Passive hides the actor.

| Pattern | Fix |
|---------|-----|
| "X was built" | Name who built it (Mike spent 16 years building it). |
| "It is believed that" | Name who believes it. |
| "The decision was reached" | Name who decided. |

## Rhetorical Setups

Announcing insight instead of delivering it.

- "What if [reframe]?" — cut, make the point.
- "Here's what I mean:" — redundant, cut.
- "Think about it:" — condescending, cut.
- "And that's okay." — unnecessary permission, cut.

## Sentence Starters to Avoid

- Sentences starting with What / When / Where / Which / Who / Why / How as a setup → restructure, lead with the subject or name the specific thing. "What makes this hard is…" → "The constraint is…"
- Paragraphs starting with "So" → start with content.
- Sentences starting with "Look," → remove.

## Dramatic Fragmentation (context-dependent)

Manufactured profundity through fragments: "This unlocks something. Power." → write a complete thought.

**Watt exception:** Jared's signature beat is fragments doing rhythmic work — "You ship. You hit PMF. Revenue compounds." Keep that. The test: is the fragment carrying real content in his cadence (keep), or is it a one-word fake-reveal for drama (cut)?

## Rhythm Patterns

| Pattern | Fix |
|---------|-----|
| Three-item lists everywhere | Use two, or one. |
| Question answered immediately | Let it breathe or cut it. |
| Every paragraph ends on a punchy one-liner | Vary endings. (Jared's forward-beat *closer* at the end of a whole post is fine; every paragraph doing it is slop.) |
| Em-dashes | Gone. See top of file. |
| Lazy extremes (every, always, never, everyone, nobody) | Use specifics. |
