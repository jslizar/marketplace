# systems-thinking

Systemic root-cause analysis. When a problem keeps coming back no matter how many times you fix it, this skill walks you through the **Questions of a Systems Thinker** and builds **Causal Loop Diagrams** to expose the structure producing the behavior — reinforcing loops, balancing loops, delays, and the leverage points that actually change outcomes.

## Skills

| Skill | What it does |
|---|---|
| `systems-thinking` | Guided systemic analysis: frame the problem, ask the systems questions, map causal loops, find leverage points |

No setup, no MCP dependencies.
