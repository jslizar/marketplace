# LinkedIn Strategy

A LinkedIn performance advisor. Ask it "why is my engagement down", "should this post have an image", "how often should I post", "is this topic worth posting about", "my post flopped" — it answers from research findings, not vibes.

## What it knows

Distilled from Virio's research on 226,000 LinkedIn posts (236 statistically validated findings, tested on held-out data), plus the public sources behind LinkedIn's 2025–2026 feed rebuild (LinkedIn Engineering blog, arXiv 2602.12354, arXiv 2510.14223). Covers:

- The four performance states (Slump / Cold Plateau / Warm Growth / Hot Streak) and how to play each
- Image effects by follower band (the U-shaped curve)
- How the six reaction types work and what each signals
- Topics that win and lose, post structure, timing and frequency
- High-frequency posting rules, steady-growth vs. moonshot playbooks
- How LinkedIn's feed actually works (2026 update)

It's honest about the big one: content choices explain ~5–6% of a single post's performance — the leverage is in the ~8× gap between the worst and best format decisions, compounded over many posts.

## Install

One skill (`linkedin-strategy`), one reference file. No dependencies, no network calls, no data connections.
