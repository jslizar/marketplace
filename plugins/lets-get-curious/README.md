# lets-get-curious

Turns a topic you *have* to post about but don't care about into one you're curious enough to write well. Runs the topic through 11 curiosity lenses (Origin, Friction, Money, Heresy, Insider, Scale, Adjacent, First-principles, Trajectory, Character, Stakes), triages the 2–3 with the most juice, then digs up specific, verified, surprising facts. The deliverable is an angle + nuggets + a way in — not a research summary.

## Modes

| Mode | When |
|---|---|
| Quick spark | You need one hook, fast — no searching |
| Deep dig (default) | Full lens run + web-verified facts |
| Simulate | "Why would anyone care?" — walk the stakes |
| Zoom out | The topic is too small; find the bigger frame |

## Setup

Web search only (any connected search tool works). No other dependencies.

Optional companions from this marketplace: hand the angle to the `watt` plugin (`watt:post`) to draft it in the founder voice, or to `dm-master` to turn it into outreach.
