# Format spec template

One file per format, saved as `abm-library/<kebab-name>.md`.

```markdown
# <Format name>

- **Archetype**: <one line — e.g. executive rise deep-dive>
- **Funnel stage**: BOFU — always. ABM content is bottom-funnel account targeting by definition; if a format can't be aimed at a named target account, it doesn't belong in this library.
- **ABM mechanic**: <who/what it targets and how it flatters or engages them — the reason the target account amplifies it>
- **Platform**: LinkedIn | X
- **Source examples**: exactly 2, refreshed by the scheduled library-refresh task from the user's curated post bank. User-pinned examples are marked `(pinned)` and never auto-replaced.
  - <URL> — <author>, <reactions> reactions / <comments> comments, <date> (pinned?)
- **Hook pattern**: <the first 1–2 lines as a fill-in template, e.g. "How did {person} go from {humble role} to {impressive role} at {company} in just {n} years?">
- **Skeleton** (block sequence):
  1. <hook block>
  2. <credibility/stat arrows block ("→ stat" lines)>
  3. <numbered body ("Here are N plays that made it work:")>
  4. <quote or human moment>
  5. <respect/congrats closer>
  6. <P.S. with CTA or hiring/link note>
- **Rhythm & formatting**: <line-break density, arrow bullets, numbered sections, length in words>
- **Why it works**: <2–3 sentences — the psychological engine: borrowed status, tag-bait, insider detail, news peg>
- **Adaptation notes**: <what must change per client; what must NOT change or the format breaks>
- **Watch-outs**: <where imitations of this format fail — e.g. picking a person nobody aspires to be, listing plays with no insider specifics>
```

Length target: the whole spec under 400 words. Capture shape, not prose.
