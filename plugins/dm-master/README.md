# dm-master

LinkedIn outreach ghostwriting. Paste a prospect's profile (or screenshot) and get the full outreach arc — connection request notes, first DMs, cold-go follow-ups, replies, call asks, and rejection comebacks — written as the client, in a register matched to the prospect.

Every run starts with a 5-dimension profile analysis and a two-axis tone read, drafts options per register, and self-checks against a banned-phrases list. The one rule that beats all others: never invent a profile detail.

## Skills

| Skill | What it does |
|---|---|
| `dm-master` | The whole arc: analyze profile → read tone → draft register-matched outreach options → self-check |

## Setup

None. Pure prompt skill — no MCP, no scripts. The `references/clients.md` file is a template: fill in your clients' voices and commonalities to ground the ghostwriting, and log what lands in `references/feedback-log.md` so proven patterns graduate into `examples.md`.

Pairs well with `lets-get-curious` (find the angle first, then hand it here for outreach).
