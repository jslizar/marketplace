## Stage directive

- **Job**: Make the problem urgent and the approach credible for evaluators. Educate the ICP on the problem space with real standing.

- **CTA**: Soft — a resource, a comment prompt, a gated asset. No demo/call asks.

## Admission test

Would someone actively evaluating this problem space save or request this? If it's pure reach-bait with no operational substance, it fails.
