# Obsession teardown drop

- **Archetype**: Obsessive deep-dive teardown of a beloved brand, given away
- **Funnel stage**: TOFU — earns reach with the ICP; the beloved brand is the draw, not the client. (Boundary case: the comment gate is a MOFU mechanic — see watch-outs.)
- **Stage mechanic**: The famous brand's gravity ("Apple", "$0 to $210B") pulls a cold audience; the visible obsession ("19 hours", "EVERY historical homepage") makes the teardown feel priceless; giving it away flips the value equation ("people say I should charge for these — nah").
- **Platform**: LinkedIn
- **Source examples**: exactly 2, refreshed by the scheduled library-refresh task from the viral_posts_all corpus.
  - https://www.linkedin.com/posts/joshlowman_yo-we-spent-the-last-19-hours-breaking-down-activity-7239273746467823616-e6T4 — Josh Lowman (iPhone 16 funnel), 2,784 reactions / 8,505 comments, 2024-09-10
  - (second example: to be filled by the scheduled refresh)
- **Hook pattern**: "Yo. We spent the last {obsessive hours} breaking down {beloved brand's} {system}."
- **Skeleton** (block sequence):
  1. Casual-obsessive hook (the hours + the subject)
  2. The subject's gravity stats ($0 → $210B, 285M visitors/month)
  3. "How'd they do it?" + checklist (✅) of the system's pillars
  4. What's inside the file: arrow list of components
  5. The completist flex ("EVERY historical homepage over 17 years")
  6. Value flip ("People say I should charge for these. Nah - I'm giving it away.")
  7. Gate: comment "{keyword}" (+ must-be-connected)
- **Rhythm & formatting**: 150–200 words. "Yo." register — casual confidence. Checklists and arrows. "All killer no filler" sign-off energy.
- **Why it works**: Borrowed brand gravity + visible labor + generosity. The obsessive completeness is the credibility; nobody doubts a person who screenshotted 17 years of homepages.
- **Adaptation notes**: Tear down the brand the client's ICP worships (for CFOs: Berkshire's letters; for PMs: Linear's onboarding). The artifact must actually be deep — the flex sets the bar.
- **Watch-outs**: This is the TOFU→MOFU bridge format: the comment gate technically breaks the TOFU no-CTA rule. Run it when reach AND capture are both goals; swap the gate for an open link when the client's TOFU directive is strict. Thin artifact + big flex = public letdown.
