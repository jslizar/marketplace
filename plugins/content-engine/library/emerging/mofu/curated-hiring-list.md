# Curated hiring list

- **Archetype**: curated-resource-list
- **Funnel stage**: MOFU — pure utility builds save-and-return trust; the newsletter CTA converts it.
- **Stage mechanic**: A themed, numbered roster (35 tiny funded teams, all hiring) where every line is company + one-line what-they-do + location. The theme is the hook; the completeness is the proof of work; links live in comments to feed the algorithm. Repeatable weekly — the format is a franchise.
- **Platform**: LinkedIn
- **Source examples**: (this week's scan, second pass — author-seeded)
  - https://www.linkedin.com/posts/benmlang_tiny-teams-now-hiring-new-list-of-35-funded-activity-7481348942643949569-MTh7 — Ben Lang, 415 reactions / 49 comments / 52 reposts, 2026-07-10 (scanned 2026-07-16)
  - Same-week variant: "Companies that just raised $10-50M (HIRING LIST)" — 264 reactions, 37 reposts, 2026-07-15 (below floor, listed for the franchise pattern)
- **Hook pattern**: "{THEME} NOW HIRING\n\nNew list of {N} {qualifier} startups, all hiring:"
- **Skeleton** (block sequence):
  1. All-caps theme hook + count promise
  2. The numbered list — company, one-line descriptor, location(s)
  3. "Direct links in the comments" (engagement + dwell mechanic)
  4. Follow CTA framed as more-of-this ("lists like this every week")
  5. Source credit (tool/partner that helped build the list)
  6. Community/newsletter CTA
- **Rhythm & formatting**: Dense numbered lines, zero prose between them; consistent `Name - descriptor (location)` grammar. Reposts are the tell — utility formats travel by shares, not comments.
- **Why it works**: Saves and reposts signal "useful" to the algorithm; the weekly cadence turns one post into a subscription habit; the curation cost is visible, so following feels like getting free labor.
- **Adaptation notes**: The theme must match the client's ICP (accounts they want to be saved by). Sourcing credit doubles as a partnership play.
- **Watch-outs**: Quality gate the list itself — one dead or wrong entry and the trust mechanic inverts. Don't fake the count; padded lists get called out in comments.
